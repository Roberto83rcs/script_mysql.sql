# ComunicaServ

## Plataforma Web para Solicitação e Acompanhamento de Serviços Comunitários

### Sobre o Projeto

O ComunicaServ é um sistema web desenvolvido como Projeto Integrador do curso de Análise e Desenvolvimento de Sistemas. O objetivo da plataforma é facilitar a solicitação, acompanhamento e gerenciamento de serviços comunitários, promovendo uma comunicação mais eficiente entre usuários e prestadores de serviços.

A solução foi concebida utilizando conceitos de Engenharia de Software, Experiência do Usuário (UX), Arquitetura de Sistemas e Modelagem de Dados.

---

## Objetivos

* Facilitar a solicitação de serviços comunitários.
* Permitir o acompanhamento do andamento das solicitações.
* Centralizar informações em uma única plataforma.
* Melhorar a comunicação entre usuários e prestadores de serviço.
* Promover organização e transparência nos atendimentos.

---

## Funcionalidades

* Cadastro de usuários
* Login e autenticação
* Solicitação de serviços
* Consulta de histórico de solicitações
* Visualização de notificações
* Gerenciamento de perfil do usuário
* Acompanhamento de status das solicitações

## Arquitetura do Sistema

O projeto utiliza uma arquitetura composta por:

* Front-end responsável pela interface do usuário.
* Back-end responsável pelas regras de negócio.
* Banco de dados MySQL para armazenamento das informações.
* Integração com serviço de notificações por e-mail.
